# 작업 요약 (Codex)

## 사용자 프론트(user-front)
- 허브 메뉴 추가: 로그인 후 차량 예약 / 내 예약 보기 / 가격 보기 선택.
- 예약 상세 화면: 배터리·남은 시간 등 데모 정보와 예약 삭제/복귀 동작.
- 헤더 개선: 로그인 이메일 인사말, 위치 기반 날씨(Open-Meteo) 표시.
- 한글 깨짐 복구: 스텝 라벨, 예시 번호판, 에러 메시지 등 UTF-8 정리.

## 관리자 프론트(admin-front)
- 세션 카드 상태 확장: 입차 대기/조기 입차/입차 확인 필요/충전 중/완료/취소 배지와 서브텍스트.
- 입차 상태 컬럼/카드에 감지 여부 표시(의사 랜덤).
- 레이아웃 토글 추가: 2열 카드 보기 ↔ 1열 리스트 보기.
- 한글 텍스트 복구 및 버튼/헤더 정리.

## 백엔드/워커
- 변경 없음(구동 스크립트 `run.ps1` 사용, OPENAI 키 자동 로드).
